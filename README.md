# FallingWords
AND ANOTHER SCHOOL PROJECT...

## Please start playing with menu.html.

## Gameplay:
Try to type the words before they fall!

### Features:
- Multiple levels
- Power-ups shop
- Multiple fall styles
- Dynamic speed
- And more!
